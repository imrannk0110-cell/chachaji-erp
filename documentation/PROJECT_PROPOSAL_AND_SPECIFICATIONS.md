# Project Proposal & Technology Specification

**Client / Brand Name:** Humjoli Ethnic  
**Project:** Custom Enterprise Resource Planning (ERP) & CRM System  
**Deployment Model:** 100% On-Premise / Localhost (Zero Cloud Dependency)  

---

## 1. Project Overview & Brand Vision

The vision for **Humjoli Ethnic** is to establish a robust, technology-driven foundation that scales with the brand's growing reputation. This project delivers a tailor-made, end-to-end ERP and CRM solution designed exclusively for the premium ethnic wear and custom tailoring business. 

By digitizing everything from order measurements and workshop tracking to financial ledgers and customer relations, this system empowers Humjoli Ethnic to operate with complete transparency, zero data loss, and high-end professionalism—matching the premium quality of the brand itself.

---

## 2. Detailed Scope of Work & Phase-Wise Checklist

As per the core client requirement, this system has been uniquely engineered to run **entirely on local hardware (Localhost)**. This strategic architectural decision completely eliminates the need for expensive monthly/yearly cloud hosting, domain renewals, or database server costs.

### Phase 1: POS & Customer Digitization [Completed]
- [x] **Smart Point of Sale (POS):** Custom order creation for ethnic wear (Sherwani, Coat, Pant, Safa, etc.) with real-time billing.
- [x] **Auto-Costing Engine:** Automatic calculation of raw material (Fabric) landing costs and Stitching profit margins.
- [x] **Digital Measurement Book:** Lifetime secure local storage of exact customer measurements.
- [x] **Advance & Balance Ledger:** Automated tracking of advance payments and "Market Udhaari" for pending balances.

### Phase 2: Workshop & Operations Pipeline [Completed]
- [x] **Manager Allocation:** Assign specific garments to specific workshop managers.
- [x] **Smart Handover Tracking:** Monitor "Ready", "Pending", and "Overdue" items dynamically based on system dates.
- [x] **Partial Delivery Split-Logic:** Intelligently split pending items into new sub-orders to ensure timely partial deliveries without confusing the ledger.

### Phase 3: Financial & Supplier Ledgers [Completed]
- [x] **Daily Daybook (Galla):** 100% automated logging of POS income, manual income, supplier payments, and shop expenses.
- [x] **Supplier Management:** Track fabric/thaan purchases, GST calculations, and pending market payables (Market Dena).
- [x] **Manager Ledgers:** Automatically track stitching credits earned vs. advance cash given to tailors.

### Phase 4: Business Analytics & CRM Automation [Completed]
- [x] **Live KPI Command Center:** Real-time dashboard showing "Aaj ka Galla", "Today's Orders", and Cash Flow ratios.
- [x] **Proactive CRM:** Automated home-screen alerts for "Today's Birthdays" and "Upcoming Festivals".
- [x] **1-Click WhatsApp:** Direct WhatsApp integration for sending personalized greetings or payment reminders.

## 3. Core Technical Infrastructure

The system is engineered using industry-standard, high-performance web technologies configured specifically for an offline retail environment.

* **Frontend Engine:** React.js powered by Vite. Features a premium "FinTech-style" responsive interface with Dark/Light Mode capabilities.
* **Backend Engine:** Node.js & Express.js REST API.
* **Database Engine:** SQLite 3. Chosen specifically to fulfill the requirement of **Zero-Configuration Local Hosting**. It acts as a highly reliable, portable database that lives entirely on the shop's hard drive.
* **Network Controller:** Custom LAN configuration allowing any mobile device connected to the shop's Wi-Fi to access the ERP securely via a QR code.
* **Disaster Recovery:** 1-Click `.db` file backup system for instant recovery on any new PC in case of hardware failure.
